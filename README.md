# ☕ Primeiros Passos com Java

Este repositório contém meus primeiros exercícios práticos em Java, focados em lógica de programação e nos pilares da **Programação Orientada a Objetos (POO)**.

### 🧠 O que pratiquei aqui:
- **Encapsulamento:** Uso de atributos privados e métodos getters/setters.
- **Lógica de Negócio:** Métodos para calcular médias, descontos e validações de idade.
- **Interação entre classes:** Criação de objetos e execução na classe Principal.

### 📂 Estrutura das Classes:
- `Aluno.java`: Cálculo de média e controle de notas.
- `Livro.java`: Exibição de detalhes e organização de atributos.
- `Produto.java`: Lógica para aplicação de descontos.
- `ContaBancaria.java` e `IdadePessoa.java`: Prática inicial de atributos e métodos.
